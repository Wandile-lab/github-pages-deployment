# GitHub Pages Deployment with GitHub Actions

> Automated deployment pipeline: push → deploy → live site in seconds.

---

## Overview

This project demonstrates a simple CI/CD pipeline that automatically deploys changes to `index.html` to GitHub Pages using GitHub Actions.

---

## How It Works

The workflow is triggered when:

- A push is made to the `main` branch  
- AND the `index.html` file is modified  

### Deployment Flow:

1. Code is pushed to GitHub  
2. GitHub Actions workflow runs  
3. Repository is checked out  
4. Files are uploaded as an artifact  
5. Site is deployed to GitHub Pages  

---


## Usage

1. Clone the repository:
2. Make changes to `index.html`
3. Push changes:
4. The workflow will automatically deploy your site
